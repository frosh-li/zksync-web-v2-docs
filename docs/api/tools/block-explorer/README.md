# Block Explorer

With zkSync [block explorer](https://explorer.zksync.io/), you can retrieve comprehensive insights or search for a particular piece of information like transactions, blocks, wallet or contract address on the zkSync network, verify your smart contracts, utilize the zkEVM Debugger to replay the execution of your smart contracts, and read each instruction.
